1. Discuss the scope of var, let, and const.

'var' declarations are scoped either globally or function scoped. 'let' and 'const' declarations are limited to the block in which they are declared. 'var' variables can be updated and re-declared within its scope, 'let' variables can be updated but not re-declared.

2. Tell us the use cases of null and undefined.

1. 'null' is used to represent an intentional absence of any value or object.
2. 'undefined' is used :

  • When a variable has not been assigned a value,
  • Function without a return value,
  • Missing function parameters,
  • Array index that doesn't have value,
  • Accessing an object property that hasn't been defined etc.

3. What do you mean by REST API?

REST API stands for Representational State Transfer Application Programming Interface. It is an architectural style for designing networked applications. It is a set of rules that define how applications or devices can connect to and communicate with each other.
